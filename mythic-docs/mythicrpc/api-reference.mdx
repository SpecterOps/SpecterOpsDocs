Use the navigation on the left to explore different categories of functions:

- **Agent Storage** - Store and manage arbitrary data
- **Authentication** - API token management  
- **Artifacts** - Create and search artifacts
- **Callback Management** - Manage callbacks and commands
- **File Management** - Handle file operations
- **Task Management** - Create and manage tasks
- And many more...

Each endpoint includes detailed request/response schemas, examples, and an interactive playground for testing.
